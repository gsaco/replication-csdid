## Repo Structure

- `python_replication.py`: main script that reproduces tables, figures, and `python.html`.
- `python_replication_jupytext.py`: notebook source (paired with `python_replication_jupytext.ipynb`).
- `replication_python.html`: standalone HTML report for GitHub Pages.
- `python.html`: HTML report generated by the script.
- `data/`: input data (county-level mortality + covariates).
- `tables/`: generated CSV tables (including `honestdid_placeholder.csv`).
- `figures/`: generated PNG figures.
